Why use OpenSource SDN

Use case driven SDN

Network Operators of various verticals from Small Enterprises to Service Providers contribute use cases and requirements to Open Networking Foundation (ONF). To better understand requirements and foster growth, ONF helps develop open source software for development to deployment.

Well Documented

All code is well documented and architected in a way that developers can pick up easily.

Rapid Deployment

Code developed is expected to be easily deplpoyed in formats applicable to the use case. Virtual machine images, container instances, Python pip style packages are some common formats that deployers can expect from projects hosted on this site.
